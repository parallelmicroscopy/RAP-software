# Starting Up

## Prerequisites
- Node.js ≥16
- Java ≥17
- Vimba SDK installed

## Quickstart
1. Clone the repo
   ```bash
   git clone https://github.com/your-org/your-repo.git
   cd your-repo

## Install dependencies
```
npm install

python3 -m venv venv
source venv/bin/activate        # or `venv\Scripts\activate` on Windows
pip install -r requirements.txt
```

## Run tests
```
npm test
pytest -q
```
